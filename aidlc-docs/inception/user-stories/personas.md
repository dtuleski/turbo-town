# User Personas

## Overview
This document defines the user personas for the memory game application. Personas represent distinct user types with specific goals and needs.

---

## Persona 1: Free User (Emma)

**Role**: Free Tier Player

**Primary Goals**:
- Play casual memory games for entertainment
- Practice memory skills without financial commitment
- Explore the game before deciding to upgrade

**Characteristics**:
- Uses free tier with basic shapes theme
- Limited to 3 games per 24 hours
- Maximum 12 pairs per game
- No access to game history, leaderboards, or achievements

---

## Persona 2: Light Tier User (Jake)

**Role**: Light Subscription Player

**Primary Goals**:
- Play memory games regularly with themed content
- Track personal progress and compete on leaderboards
- Access premium themes at affordable price

**Characteristics**:
- Pays $1.99/month (or annual equivalent)
- Up to 10 games per day
- Maximum 24 pairs per game
- Access to all themes, game history, leaderboards, and achievements

---

## Persona 3: Standard Tier User (Sophia)

**Role**: Standard Subscription Player (Recommended Tier)

**Primary Goals**:
- Play memory games frequently with varied difficulty
- Compete seriously on leaderboards
- Earn achievements and track detailed statistics

**Characteristics**:
- Pays $5.99/month (or annual equivalent)
- Up to 30 games per day
- Maximum 36 pairs per game
- Access to all themes, game history, leaderboards, and achievements

---

## Persona 4: Premium Tier User (Marcus)

**Role**: Premium Subscription Player

**Primary Goals**:
- Play unlimited games with maximum difficulty
- Dominate leaderboards across all themes
- Complete all achievements and maximize scores

**Characteristics**:
- Pays $9.99/month (or annual equivalent)
- Up to 100 games per day
- Maximum 48 pairs per game
- Access to all themes, game history, leaderboards, and achievements

---

## Persona 5: Admin User (Alex)

**Role**: System Administrator

**Primary Goals**:
- Manage user accounts and subscriptions
- Create and manage theme content through CMS
- Monitor system health and analytics
- Handle user support issues

**Characteristics**:
- Full access to admin dashboard
- Can perform CRUD operations on themes
- Views analytics and user metrics
- Manages subscription modifications and refunds

---

## Persona 6: Content Manager (Riley)

**Role**: Theme Content Creator

**Primary Goals**:
- Create and publish new game themes
- Update existing theme content
- Preview themes before publishing
- Manage theme metadata and images

**Characteristics**:
- Access to Content Management System
- Can upload images and define matching pairs
- Can enable/disable themes
- Limited admin access (content-focused)

---

## Persona Mapping to User Stories

- **Emma (Free User)**: Registration, basic gameplay, tier limitations, upgrade prompts
- **Jake (Light Tier)**: Subscription purchase, themed gameplay, leaderboard viewing, basic achievements
- **Sophia (Standard Tier)**: Advanced gameplay, competitive leaderboards, comprehensive achievements
- **Marcus (Premium Tier)**: Maximum difficulty gameplay, leaderboard domination, achievement completion
- **Alex (Admin)**: User management, system monitoring, subscription management, analytics
- **Riley (Content Manager)**: Theme creation, content publishing, CMS operations
